# Drawing-pictures

An iOS application that draw selected drawing.

## What I Learned
 - layout from code
 - custom hitTest-ing
 - save images to device
 - supporting differnt IOS operation systems
 - UI elements: `UINavigationController`, `CALayer`, `UIBezierPath`.
